Limbdark.jl Documentation

Fast, analytic, and numerically stable transit light curves for stars with arbitrary order limb darkening. Check out the latest draft of the paper here. The code is open source and available on GitHub.

This documentation is under construction and is still being updated with examples, tutorials, and detailed API documentation. If you don't see what you're looking for, feel free to open an issue.

Check out the Installation, Quick start, and API pages for more information.
